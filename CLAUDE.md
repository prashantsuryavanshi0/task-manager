# AI Guidance File

## Purpose

This project used AI tools to assist in development while maintaining code quality and correctness.

---

## AI Usage

- Used AI for generating boilerplate code
- Used AI for debugging errors
- Used AI for UI styling suggestions

---

## Constraints

- All generated code was manually reviewed
- No sensitive or confidential data was used
- Followed simple and readable coding practices

---

## Coding Standards

- Clean and readable code
- Proper separation of frontend and backend
- Simple API structure

---

## Risks & Mitigation

- Risk: Incorrect AI-generated code
- Mitigation: Manual testing and verification

---

## Conclusion

AI was used as a helper tool, not as a replacement for understanding the code.
